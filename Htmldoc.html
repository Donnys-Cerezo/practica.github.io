Introducción

El Lenguaje de Marcado de Hipertexto (HTML) es la base fundamental para la creación de páginas web. Desde su creación en los años 90, HTML ha evolucionado para convertirse en un lenguaje esencial en el entorno digital. Su función principal es organizar y estructurar el contenido de una página web, incluyendo textos, imágenes, enlaces y multimedia, utilizando etiquetas o "tags" que definen cómo se muestran los elementos en el navegador. Este proyecto explorará en profundidad HTML, sus características, estructura básica y etiquetas clave, como encabezados, párrafos, enlaces, imágenes y tablas. También se analizará su evolución para adaptarse al desarrollo web moderno y su importancia en la creación de sitios accesibles, eficientes y optimizados para motores de búsqueda (SEO). El objetivo es ofrecer una comprensión completa de HTML y su papel crucial en la creación y mantenimiento de páginas web.

HTML

HTML son las siglas en inglés de Hyper Text Markup Language o Lenguaje de Marcas de Hipertexto. Es un lenguaje de programación que utiliza etiquetas para definir las distintas partes de una página web, desde el texto hasta las imágenes. Para ser ejecutado o interpretado, se necesita usar un navegador de internet, como Internet Explorer, Google Chrome, Firefox, entre otros.

Su utilidad radica en el control total que ofrece al diseñador, permitiendo superar la simplicidad y hacer el contenido más dinámico y versátil. HTML se usa no solo para crear la estructura de una página web, sino también para la creación de hipervínculos, imágenes y videos, haciéndola más llamativa e interactiva.

Tags de HTML

¿Qué definen la estructura de una página web?

Toda página web está estructurada con los siguientes elementos:

  • <Doctype>: Define el tipo de documento y la versión de HTML.
  • <html></html>: Indica que se está usando HTML.
  • <head></head>: Representa el encabezado de la página web.
  • <meta>: Añade meta-información al documento HTML.
  • <body></body>: Contiene el contenido visible del documento.

Elementos comunes de HTML

Elementos del <head>:

  • <title></title>: Define el nombre de la página, visible en la pestaña del navegador.
  • <link>: Enlaza documentos CSS, recursos web o iconos.
  • <style></style>: Permite incluir CSS directamente en el documento HTML.
  • <script></script>: Sirve para enlazar lenguajes de programación como JavaScript.
  • <base>: Define una base para los enlaces en la página.

Elementos del <body>:

  • <header></header>: Incluye logotipos, títulos o menús de navegación.
  • <nav></nav>: Define enlaces de navegación.
  • <main></main>: Contiene el contenido principal de la página web.
  • <article></article>: Define un artículo dentro de la página.
  • <section></section>: Agrupa contenido relacionado en una sección.
  • <aside></aside>: Contenido relacionado de forma indirecta con el contenido principal.
  • <footer></footer>: Define el pie de página con información de contacto.
  • <div></div>: Contiene partes del documento HTML, útil para organizar contenido.
  • <h1></h1> a <h6></h6>: Encabezados con diferentes niveles de importancia.
  • <p></p>: Define un párrafo de texto.
  • <a></a>: Crea enlaces a otras páginas web.
  • <ul></ul>: Crea una lista no ordenada.
  • <ol></ol>: Crea una lista ordenada.
  • <li></li>: Define un elemento de lista.
  • <form></form>: Utilizado para crear formularios.
  • <table></table>: Crea una tabla.
  • <video></video>: Permite añadir videos a la página.
  • <img>: Permite añadir imágenes al documento HTML.
  • <audio></audio>: Permite añadir audios a la página web.

Conclusión

En resumen, HTML ofrece una estructura sólida y versátil para definir el contenido y la funcionalidad de una página web. A través de diferentes etiquetas, facilita la organización del contenido y la integración de elementos multimedia como imágenes, videos y enlaces, mejorando la experiencia del usuario.
